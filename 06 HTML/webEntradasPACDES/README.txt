# PAC de Desarrollo de Lenguajes de Marcas DAW

## Resumen
Este es el ejercicio de Desarrollo que realizamos para la asignatura de Lenguaje de Marcas del grado superior "desarrollo de aplicaciones web".

Consiste en:
- Una página donde seleccionar:
    - Un día y lugar de un concierto.
    - Tambien el número de entradas y la categoría de la misma.
    - Seleccionar métodos de pago y aceptar condiciones.
- Una página de confirmacion de compra donde se muestre los datos recogidos en el formulario anterior.
